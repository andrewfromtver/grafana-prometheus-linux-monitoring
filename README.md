# how to

* Add `.env` file (in `./` folder)
* Execute command`docker compose -p monitoring up -d --wait grafana` (in `./` folder)

# .env params

*Grafana creds*
* `GF_SECURITY_ADMIN_USER=admin`
* `GF_SECURITY_ADMIN_PASSWORD=qwerty12`

# http page

*Grafana web UI*
`http://localhost:3000`
